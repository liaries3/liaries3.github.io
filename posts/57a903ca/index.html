解决Github下载慢、访问慢的问题

2021-06-19
352- 1m-
Github > Github下载慢-Github

前言

无法保证百分之百有效，但应该是能从一定程度上缓解这些问题。

  作为一个才接触Github不久的人，每次想要用Github都十分艰难，无论是进网页还是下载项目，速度真的是太太太太慢了！

下载慢

前提：

  • 可安装扩展的浏览器，如谷歌浏览器
  • 浏览器插件，Github加速

具体方法

  1. 浏览器网址栏输入chrome://extensions/，按回车
  2. 打开右上角的开发者模式
  3. 将下载好的插件压缩包解压出来
  4. 找到github加速.crx，将其拖入到浏览器中，根据提示完成安装。（注：Github加速.crx文件请勿从电脑上删掉）
  5. 随意找一个Github项目，看到有加速按钮即安装成功！

访问慢

主要参考了Github520项目。

具体步骤：

  1. 下载SwitchHosts
  2. 点击左上角的+，弹出添加hosts窗口
  3. Hosts类型选择远程
  4. Hosts标题随意填写
  5. URL填入https://raw.fastgit.org/521xueweihan/GitHub520/main/hosts
  6. 刷新时间选择1小时，点击确定
  7. 点击右上角的设置，选择选项
  8. 在选项面板点击命令
  9. 在Hosts 应用后命令的框里填入ipconfig /flushdns，点击确定
  10. 软件挂在后台就会自动更新电脑的hosts文件，并自动刷新缓存，能从一定程度上缓解加载慢的状况。
